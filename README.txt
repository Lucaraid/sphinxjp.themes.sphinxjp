A sphinx theme for sphinx-users.jp site.

Output Sample
==============
:output: http://packages.python.org/sphinxjp.themes.sphinxjp/

Features
========
* provide ``sphinxjp`` theme used on the sphinx-users.jp site
  ( http://sphinx-users.jp/ ).

Setup
=====
Make environment with pip install::

    $ pip install sphinxjp.themes.sphinxjp


Convert Usage
==============
setup conf.py with::

    html_theme = 'sphinxjp'

and run::

    $ make html


Requirements
============
* sphinx 1.2 or later
